Filtros CSS

Los filtros CSS son efectos visuales que podemos aplicar a imágenes y elementos HTML. Aquí veremos algunos de los filtros más comunes y útiles.

Blur (Desenfoque):

El filtro blur desenfoca la imagen. Se especifica en píxeles (px). A mayor valor, mayor será el desenfoque. En este caso usamos 5px.

Brightness (Brillo):

Brightness controla el brillo. 100% es normal, mas de 100% aumenta el brillo, menos de 100% lo reduce. Aqui usamos 150% para hacerla mas brillante.

Grayscale (Escala de grises)

Convierte la imagen a blanco y negro. 100% es totalmente en escala de grises, 0% mantiene los colores originales.

Sepia

Da un tono marrón antiguo a la imagen, se usa para darle un efecto vintage. 100% aplica el efecto completamente.
