Propiedades CSS para Imagenes

Las propiedades CSS para imagenes nos permiten controlar como se muestran y ajustan las imagenes en nuestras paginas web. La propiedad object-fit es especialmente util para controlar como una imagen debe redimensionarse para ajustarse a su contenedor.

Object-fit: Cover

La imagen mantiene su relación de aspecto y cubre todo el contenedor, recortando las partes que sobresalen. Es util cuando queremos que la imagen llene completamente su espacio sin deformarse.

Object-fit: Contain

La imagen mantiene su relación de aspecto y se ajusta dentro del contenedor sin recortes. Puede dejar espacios en blanco pero muestra la imagen completa.

Object-fit: Fill

La imagen se estira para llenar el contenedor, sin mantener su relación de aspecto. Puede resultar en una imagen distorsionada pero ocupara todo el espacio disponible.

Object-fit: Scale-down

Similar a contain, pero nunca aumentara el tamaño de la imagen mas alla de sus dimensiones originales. Util para evitar que imagenes pequeñas se vean pixeladas.
